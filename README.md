# wave
A flexible audio API for ComputerCraft

Features:
- Support for Command Computer and Iron Note Block ([MinimalPeripherals](https://github.com/justync7/MinimalPeripherals)) outputs
- Support for Note Block Studio tracks
- Instrument filters (for example: bass + base drum on one output, other instruments on another)
- Multiple track playback at the same time
- Volume control
- Throttling

For more information, visit the [wiki](https://github.com/CrazedProgrammer/wave/wiki).
